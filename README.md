# Генератор ТЗ для курсового проекта

## Описание

Каталог содержит исходные файлы технических заданий для курсового проекта 3 курса. Структура документов ориентирована на ГОСТ 19.201-78 и адаптирована под учебный проект по программной инженерии.

---

## Порядок использования

1. Установите [плагин Typst](https://marketplace.visualstudio.com/items?itemName=myriad-dreamin.tinymist) для VS Code.
2. Выберите нужный комплект ТЗ: общий документ или индивидуальный вариант.
3. Заполните исходные данные о проекте и исполнителях в `data.typ`.
4. При необходимости уточните формулировки в `body.typ`.
5. Соберите документ командой `typst compile main.typ`.

---

## Структура репозитория

- `main.typ` - точка сборки документа;
- `template.typ` - шаблон страниц;
- `cfg.typ` - параметры оформления;
- `data.typ` - сведения о проекте, исполнителях и аналогах;
- `body.typ` - основной текст технического задания.

---

## Основные разделы

- ВВЕДЕНИЕ
- ОСНОВАНИЯ ДЛЯ РАЗРАБОТКИ
- НАЗНАЧЕНИЕ РАЗРАБОТКИ
- ТРЕБОВАНИЯ К ПРОГРАММЕ
- ТРЕБОВАНИЯ К ПРОГРАММНОЙ ДОКУМЕНТАЦИИ
- ТЕХНИКО-ЭКОНОМИЧЕСКИЕ ПОКАЗАТЕЛИ
- СТАДИИ И ЭТАПЫ РАЗРАБОТКИ
- ПОРЯДОК КОНТРОЛЯ И ПРИЕМКИ
- СПИСОК ИСПОЛЬЗУЕМОЙ ЛИТЕРАТУРЫ
- ПРИЛОЖЕНИЕ (если есть)

---

## Данные в `data.typ`

- `code` - код проекта; при необходимости его можно уточнить по [классификатору программного обеспечения](https://ru-ikt.ru/reestrpo);
- `project_name` - название проекта;
- `project_name_english` - название проекта на английском языке;
- `students` - список исполнителей с указанием учебной группы;
- `agreed_by_name` - научный руководитель;
- `agreed_by_position` - должность научного руководителя.
